# uav_simulator

### Install
build the package:
```
git clone https://github.com/Zhefan-Xu/uav_simulator.git

cd ~/catkin_ws
catkin_make
```

setup environment variable. Add following to your ```~/.bashrc```
```
source path/to/uav_simulator/gazeboSetup.bash
```

### Run
```
roslaunch uav_simulator start.launch
```
